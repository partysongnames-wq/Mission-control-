Taskboard Suite

Every task you assign is a guest at the bar. Track status live, see assignments, and watch progress drift from backlog to review to done. This is the command center for visibility.

openclaw skill run mission-control-taskboard
Automatic heartbeat checks keep the taskboard updated. Reverse prompt: “What tasks should we prioritize based on current projects?”

Calendar Panorama

View every cron job, scheduled routine, and heartbeat nudge in one elegant calendar. The Standard schedule deck, but for OpenClaw—see proactively what’s running tomorrow at 6:30 or tonight at 20:00.

openclaw skill run mission-control-calendar
Ask: “Show my upcoming cron runs.” This screen proves your agent is truly proactive.

Project Lobby

Celebrate your live projects like suites overlooking the skyline. Each card tells you how close you are, what tools feed in, and when to reverse prompt for your next move.

openclaw skill run mission-control-projects
Add notes: “Which project needs momentum today?” and let OpenClaw suggest the next meaningful task.
