Travel Scan

Fresh Bangkok → Japan fare intel on demand. Run the topic-monitor/Skyscanner query you already use to spot price drops before they disappear.

openclaw topic-monitor run --name "extra:skyscanner-watch" --prompt "BKK→JPN fares"

Pro tip: run this just before planning flights; compare results to earlier baseline notes in memory/2026-02-28.md.

Morning Manifesto

Draft tomorrow’s top 3 with a travel + job focus. The morning-manifesto skill already knows your resolutions—just nudge it toward the travel mindset.

openclaw skill run morning-manifesto --prompt "Travel + job focus"

Runs best right before your day starts—use it as a mantra before checking updates.

Memory Snapshot

Capture a playlist idea, travel reminder, or unexpected lesson—Supermemory is waiting. Paste your note and push it to your long-term knowledge base.

openclaw skill run supermemory --context "Mission Control note" --note "[paste fragment]"

Need a short snippet? Paste into the input, run the command, and you’ll find it later via memory search.

Travel Mission Taskboard

Turn the manifesto into live work. These three pieces are the ones you want locked before tomorrow’s departure—each tied to a taskboard update, heartbeat check, or reverse prompt so the plan stays visible.

  • Bookings locked: Confirm the flight/train + hotel/ride details, seat selections, and itinerary times.
  • Travel kit ready: Lay out clothes, chargers, toiletries, and documents; note any missing bits to grab.
  • Transit mapped: Sketch the departure-day route, last-mile options, buffers, and alarms.

These tasks live in mission-control-travel-tasks.md and are backed by tonight’s 21:00 and tomorrow’s 07:00 reminders, so keep the card visible while you tick them off.

Why this dashboard?

Mission Control is a digest of the three pathways you care about most: trip alerts, daily focus, and memory capture. Use this widget as your go-to home screen. Open it in any browser and treat it like a mini control center. Tap a card, copy the command, and drop it into Terminal whenever you want the action to happen.

Want automation? Wrap any of the commands in a cron job or shortcut—`openclaw cron add` can re-run these on a schedule, or you can launch them manually. Need help wiring that? Just say the word.
